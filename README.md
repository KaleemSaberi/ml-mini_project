# House Price Prediction Using Machine Learning

## Description
This project predicts house prices using a machine learning model.
It is a beginner-level ML mini project.

## Dataset
- Source: Kaggle / UCI Repository
- Format: CSV

## Tools Used
- Python
- Jupyter Notebook
- Pandas, NumPy
- Scikit-learn

## Steps
1. Data collection
2. Data preprocessing
3. Model training
4. Price prediction

## Results
The model predicts house prices with reasonable accuracy.
